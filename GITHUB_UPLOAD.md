2. Repository name: `abz-shutdown`
3. Description: `ABZ Shutdown - Standalone UEFI ACPI Shutdown App`
4. Visibility: Public (recommended)

### Step 2: Push Your Code

After creating the repo, GitHub shows setup commands. Use these commands in your terminal:

```bash
cd /Users/abz/Documents/Abz_Shutdown

# Add GitHub as remote
git remote add origin https://github.com/aterro/abz-shutdown.git

# Rename branch to main
git branch -M main

# Push all commits
git push -u origin main
```

### Step 3: Authenticate with Personal Access Token

When git asks for a password:

1. Go to https://github.com/settings/tokens
2. Click "Generate new token" → "Generate new token (classic)"
3. Token name: `ABZ Shutdown Upload`
4. Expiration: 90 days
5. Scope: Check ✅ "repo"
6. Click "Generate token"
7. Copy the token (shown once!)
8. Paste it when git asks for password
